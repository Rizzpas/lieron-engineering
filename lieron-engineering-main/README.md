**Lieron Engineering Limited** is a premier structural steel engineering firm based in Auckland, NZ. This repository contains our high-performance, animation-rich corporate digital experience.

---

## 💎 Premium Features

### ⚡ Mobile Performance Optimized
- **Ambient GPU Offloading**: Custom-built background engine that switches from heavy blurred orbs to lightweight gradients on mobile.
- **Smart Animation Culling**: Disables non-essential infinite loops on mobile to preserve CPU/Battery.
- **AVIF/WebP Delivery**: Automated image pipeline for sub-millisecond load times.

### 🤖 Intelligent Chat Assistant
- **High-Performance Local Engine**: Zero-latency, zero-cost AI assistant with custom intent detection.
- **Natural UX**: Simulated typing delays and interactive action buttons.
- **Streaming Ready**: Built with a modular architecture that supports OpenAI streaming integration.

### 🎨 State-of-the-Art Design
- **Glassmorphism**: Sophisticated backdrop-blur effects optimized for GPU performance.
- **Dynamic Scroll States**: Custom-built IntersectionObserver provider for synchronized entrance animations.
- **Dark Mode Native**: First-class dark theme support with no FOUC (Flash of Unstyled Content).

## 🛠️ Technical Stack

- **Framework**: [Next.js 15 (App Router)](https://nextjs.org/)
- **Styling**: [Vanilla CSS](https://developer.mozilla.org/en-US/docs/Web/CSS) + [Tailwind CSS 4](https://tailwindcss.com/)
- **Animations**: [Framer Motion 12](https://www.framer.com/motion/)
- **State Management**: React Hooks (Optimized for Client Components)
- **Deployment**: Vercel Optimized
